    Example 21.

    One last (very useful!) observation. Recall: If \(A\cap B = \emptyset\text{,}\) then \(\Pr(A\cup B) = \Pr(A) + \Pr(B)\text{.}\)
    ðŸ”—
    For any event \(A\text{,}\) \(A\cap A^c = \emptyset\) and \(A \cup A^c = \Omega\text{.}\) So:
    \begin{gather*} 1 = \Pr(\underbrace{A \cup A^c}_{\Omega}) = \Pr(A) + \Pr(A^c). \end{gather*}
    We can rewrite this in two useful ways:
    \begin{align*} \Pr(A) \amp = 1 - \Pr(A^c) \\ \Pr(A^c) \amp = 1 - \Pr(A) \end{align*}
